Project Description

This project is being developed with the goal of creating a modern and maintainable club website for the basketball department of TSV Lindau. The focus is on clear information architecture and traceable technical implementation.

Backend considerations are integrated from the beginning. A Python-based API integration is planned for automated processing of league data, along with an administrative interface for content management.

Technical Approach

  • ° semantic HTML5
  • ° CSS3 with Grid and Flexbox
  • ° Vanilla JavaScript for Frontend Interactions
  • ° The backend architecture is prepared; a first Python-based API prototype for data processing is in implementation.
  • ° Use of a backend framework is planned

Challenge & Solution

Challenge: Building a scalable website structure that cleanly separates frontend development and future backend integration.

Solution: Early definition of the page and component structure, as well as preparation of clear API interfaces, before backend functionality is implemented.
